# 📊 Bitcoin Sentiment Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing the relationship between Bitcoin market sentiment (Fear/Greed Index) and trader performance using real-world trading data. Financial markets are heavily influenced by emotions, and this project aims to quantify how psychological factors like fear and greed impact trading outcomes.

By combining sentiment data with historical trade records, the project performs a detailed analysis to understand:

* How market sentiment affects profitability

* Whether traders perform better in Fear or Greed conditions

* Behavioral patterns such as overtrading or risk-taking

The workflow includes data cleaning, merging multiple datasets, performing exploratory data analysis (EDA), and building visual insights using Python and Power BI.

The final outcome is an interactive Power BI dashboard that provides a clear and intuitive understanding of trading performance under different market sentiments. This helps in identifying data-driven strategies and improving decision-making in trading environments.

---

## 🎯 Objectives

* Analyze trader profitability under different market sentiments
* Identify hidden patterns in trading behavior
* Compare Fear vs Greed market conditions
* Generate actionable insights for decision-making
